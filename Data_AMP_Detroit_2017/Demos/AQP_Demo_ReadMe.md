Here are the instructions for demonstrating Adaptive Query Processing for SQL Server 2017:

1) Download WideWorldImportersDW-Full.bak from https://github.com/Microsoft/sql-server-samples/releases/tag/wide-world-importers-v1.0

2) Restore the database on SQL Server 2017 CTP 2.1 or greater (I have confirmed this works for RC1 as well)

3) Use SQL Server Management Studio version 17.1 or greater

4) Create the MSTFV [Fact].[WhatIfOutlierEventQuantity] from https://github.com/joesackmsft/Conferences/blob/master/Data_AMP_Detroit_2017/Demos/WhatIfOutlierEventQuantity.sql

5) Use the full demo script is here: https://github.com/joesackmsft/Conferences/blob/master/Data_AMP_Detroit_2017/Demos/AQP_AMP_Demos.sql
